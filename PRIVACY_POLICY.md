# Privacy Policy

**Last Updated:** January 15, 2026

**FocusCoach** ("we", "us", or "our") is developed by Arda Nogay, an individual developer. This Privacy Policy explains how we collect, use, and protect your information when you use the FocusCoach mobile application ("App").

By using FocusCoach, you agree to the collection and use of information in accordance with this policy.

---

## 1. Information We Collect

### 1.1 Information You Provide

| Data Type | Description | Purpose |
|-----------|-------------|---------|
| **Account Information** | Email address, display name (when signing in with Google, Apple, or Email) | Account creation and identification |
| **Task Data** | Task titles, completion status, reminder settings | Core app functionality |
| **Session Data** | Focus session durations, break times, timestamps | Productivity tracking and statistics |

### 1.2 Automatically Collected Information

| Data Type | Description | Purpose |
|-----------|-------------|---------|
| **Device Information** | Device type, operating system version | App optimization and debugging |
| **Usage Analytics** | App interactions, feature usage (anonymized) | Improving user experience |
| **Crash Reports** | Error logs and crash data | Bug fixing and stability |

### 1.3 Pro Subscription Data (Optional)

If you subscribe to FocusCoach Pro, we also collect:

| Data Type | Description | Purpose |
|-----------|-------------|---------|
| **Sync Data** | Tasks, sessions, achievements, settings | Cloud synchronization across devices |
| **AI Insights Data** | Session statistics (anonymized) | Generating personalized productivity insights |

---

## 2. How We Use Your Information

We use your information to:

- Provide and maintain the App's core functionality
- Track your focus sessions and productivity statistics
- Send local notifications for timers and reminders
- Sync your data across devices (Pro users only)
- Generate personalized productivity insights (Pro users only)
- Process subscription payments
- Improve the App through analytics
- Respond to your support requests

---

## 3. Data Storage and Security

### 3.1 Local Storage

- Your data is primarily stored locally on your device using CoreData
- Settings are stored using UserDefaults
- No account is required to use the basic features

### 3.2 Cloud Storage (Pro Users)

- Cloud data is stored in Google Firebase Firestore
- Data is encrypted in transit and at rest
- Access is restricted to authenticated users only
- Each user can only access their own data

### 3.3 Security Measures

- Industry-standard encryption (TLS 1.3)
- Secure authentication via Firebase Auth
- No storage of payment information (handled by Apple)

## 5. Your Rights and Choices

### 5.1 Access and Export

- View your data within the App
- Export your session history (Pro users)

### 5.2 Deletion

- Delete individual tasks and sessions
- Delete your account and all associated data
- Request complete data deletion by contacting us

### 5.3 Opt-Out Options

- Use the App without creating an account
- Disable notifications in device settings

## 6. Data Retention

- **Local Data:** Retained until you delete the App or clear data
- **Cloud Data (Pro):** Retained until you delete your account
- **Analytics Data:** Aggregated and anonymized; retained for 26 months
- **Support Communications:** Retained for up to 3 years
